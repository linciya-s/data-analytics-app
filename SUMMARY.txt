💾 DATA STORAGE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Location: ~/.data_analytics_app/history.json

Format: JSON
{
  "id": 1705333200000,
  "filename": "sales_data.csv",
  "file_type": "csv",
  "timestamp": "2024-01-15T14:30:00.000000",
  "rows": 5000,
  "columns": 15,
  "size_mb": 2.45,
  "numeric_cols": 8,
  "categorical_cols": 7
}

Auto-created: On first file upload
Capacity: Up to 50 records
Searchable: Simple JSON search possible
Exportable: Download as .json file
Private: Stored locally on your computer
